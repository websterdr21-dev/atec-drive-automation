<objective>
Fix the Direct ATEC Telegram bot bug where photos are saved in the site root folder instead of a unit subfolder when the user confirms the upload destination at the site root during guided navigation.

Root cause: `_handle_nav_reply` lines 931–934 explicitly handles "user pressed u at site root" by silently setting `folder_id = site_id` (the site root) and skipping `SITES.learn()`. No unit folder is ever created.

Fix: when `segments` is empty (user at site root), auto-create `Unit [unit_number]` under the site root using `_find_or_create_folder`, learn the single-segment template `['Unit {unit}']`, and use the new unit folder as the upload destination.
</objective>
